 # Relatório Automático de Vendas

 # O que faz
Lê uma planilha de vendas, calcula faturamento por loja, quantidade vendida e ticket médio, e envia um relatório por e-mail automaticamente.

 # Tecnologias
- Python
- pandas (leitura e cálculos)
- openpyxl (suporte a Excel)
- smtplib (envio de e-mail)
